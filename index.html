// Historia de GIT

1972
SCCS - El Origen
Bell Labs crea SCCS (Source Code Control System), el primer sistema de control de versiones. Solo guardaba diferencias entre archivos. El abuelo de Git!
1986
CVS Aparece
CVS (Concurrent Versions System) permite que multiples desarrolladores trabajen en el mismo proyecto. Revolucionario, pero con muchos bugs y limitaciones.
2000
SVN Mejora Todo
Subversion (SVN) arregla los problemas de CVS. Se vuelve el estandar en empresas. Pero sigue siendo centralizado: si el servidor cae, nadie trabaja.
2002
BitKeeper & Linux
El kernel de Linux usa BitKeeper, un sistema propietario. Linus Torvalds lo ama por ser distribuido. Pero en 2005, BitKeeper revoca la licencia gratuita...
2005
Nace GIT!
Linus Torvalds, furioso, crea Git en solo 2 SEMANAS! Lo disena para ser rapido, distribuido y seguro. El nombre significa "idiota" en britanico (Linus tiene humor).
2008
GitHub Revoluciona
GitHub lanza su plataforma. Git + Social Network = exito masivo. Por primera vez, compartir codigo es facil y divertido. Nace el open source moderno.
2018
Microsoft Compra GitHub
Microsoft adquiere GitHub por $7.5 BILLONES. La comunidad temia lo peor, pero GitHub mejoro. Ironia: Microsoft odiaba el open source en los 2000s.
HOY
Git Domina el Mundo
+95% de desarrolladores usan Git. GitHub tiene +100 millones de usuarios. GitLab, Bitbucket compiten. Git es EL estandar mundial para codigo.
